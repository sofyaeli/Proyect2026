Guía de Etiquetas HTML

Etiqueta Definición
<!DOCTYPE html> Define que el documento es HTML5.
<html> El contenedor raíz de una página web.
<head> Contiene metadatos e información que no se muestra directamente al usuario (título, scripts, estilos).
<meta> Define metadatos como el juego de caracteres (charset) o la configuración de la vista (viewport).
<title> Establece el título de la página que aparece en la pestaña del navegador.
<link> Relaciona el documento con recursos externos (como el favicon o archivos CSS).
<body> Contiene todo el contenido visible de la página (texto, imágenes, videos).
<p> Define un párrafo de texto.
<table> Se utiliza para crear una tabla de datos.
<tr> (Table Row) Define una fila completa en la tabla.
<th> (Table Header) Define una celda de encabezado. El texto suele aparecer en negrita y centrado por defecto.
<td> (Table Data) Define una celda de datos estándar. Es el contenedor para el contenido normal de la tabla.
